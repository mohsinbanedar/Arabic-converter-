# Video Script: How to Use Arabic Listener App (After Install)

Use this ready script to record a 1-2 minute demo video for users.

## Shot 1: App open (0:00 - 0:10)
- Show app icon and open the app.
- Voiceover (Hinglish):
  - "App install hone ke baad isko open karo. Yahan pe saved notes list dikh rahi hai."

## Shot 2: Enable accessibility (0:10 - 0:30)
- Tap **Open Accessibility Settings**.
- Go to **Installed apps / Downloaded apps** and open **Arabic Listener**.
- Turn **Use service** ON and confirm permission dialog.
- Voiceover:
  - "Real-time detection ke liye Accessibility Service ON karna mandatory hai."

## Shot 3: Live Arabic detection demo (0:30 - 1:00)
- Open any app/page with Arabic text (WhatsApp, browser, notes).
- Keep text visible on screen, for example: `شكرا` or `الدوحة`.
- Return to Arabic Listener app and show new note appearing.
- Voiceover:
  - "Jaise hi Arabic text screen par aata hai, app immediately detect karke note save karti hai."

## Shot 4: Offline proof (1:00 - 1:25)
- Turn on Airplane mode (internet OFF).
- Repeat with Arabic text again.
- Show a new note still gets added.
- Voiceover:
  - "Internet OFF hone par bhi conversion kaam karta hai, kyunki app offline-first hai."

## Shot 5: Output behavior (1:25 - 1:40)
- Open latest note and highlight English/Hinglish lines.
- Voiceover:
  - "Output mein extra explanation nahi aata. Sirf converted text hi aata hai."

## Quick Checklist (before recording)
- Accessibility service ON
- Battery optimization disabled for app (recommended)
- Arabic sample text ready in another app
- Auto-rotate OFF for stable recording
- Font size normal for readability
